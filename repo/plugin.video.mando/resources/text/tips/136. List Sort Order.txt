!!NEW!!
[B]List Sort Order[/B] (Settings > Content) sets the default order for movie and TV lists from Trakt (Watchlist / Library), Simkl status shelves, and MDBList.

[B]Per-list overrides[/B]
Use the context menu [B]Set Custom Sort[/B] on an individual list to override the default for that list only. [B]Use Default[/B] clears the override.

[B]Simkl[/B]
Each status shelf has its own sort for Movies and for TV: Plan to Watch, Watching, Completed, On Hold, and Dropped.

[B]TMDb[/B]
List [I]contents[/I] are sorted per list via [B]Set Custom Sort[/B] (Watchlist, Favorites, Recommendations, and custom My Lists). They do not use the Content > Movies/TV defaults.

[B]TMDb My Lists Display Order[/B] (same Settings page) only orders the list of list names under My Lists — not the titles inside a list.

[B]Trakt Library[/B]
Trakt renamed Collection to Library on their site. Mando menus and managers use [B]Library[/B]; the Trakt API path is unchanged.
